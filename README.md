# GDSC NIT Calicut

Official website for the **Google Developer Student Clubs** chapter at **National Institute of Technology Calicut**.

## Features

- Activities, events, and announcements sections
- Team member profiles with role highlights
- Custom futuristic cursor with trail effects
- Glassmorphism navigation and card design
- Responsive layout

## Tech Stack

- Vanilla HTML5 / CSS3
- Custom JavaScript for interactive cursor and animations
- Static assets (images, logos)

## Setup

```bash
git clone https://github.com/bip-krishna/GDSC-nitc.git
cd GDSC-nitc
# Serve locally
python3 -m http.server 8000
```

Open `http://localhost:8000` in your browser.
